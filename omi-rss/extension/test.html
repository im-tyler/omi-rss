Omi RSS Extension Test Page

Extension Installation Check

To test the extension:

  1. Open Chrome and go to chrome://extensions/
  2. Enable "Developer mode" (top right)
  3. Click "Load unpacked"
  4. Select the browser_extension folder
  5. Click the extension icon in the toolbar

Common Issues & Solutions

  • Extension not loading: Check for errors in chrome://extensions/
  • Popup not formatted: Right-click the extension icon → "Inspect popup" to see console errors
  • Icons missing: Run create_icons.ps1 to generate placeholder icons

Quick Debug

Open the browser console (F12) and run:

chrome.runtime.getManifest()

This should show the extension manifest if it's loaded correctly.
