Installation Instructions

  1. Open Chrome/Firefox extension management page
  2. Enable "Developer mode"
  3. Click "Load unpacked" and select the browser_extension folder
  4. The Omi RSS icon should appear in your toolbar
  5. Click the icon to see the beautiful glassmorphism popup
  6. Use the "Pop Out Window" button for a standalone window
  7. Use the "Open Sidebar" button for the side panel view
